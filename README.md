# MCP-Use Local LLM

- Local LLM loader (HuggingFace Transformers/vLLM supported).
- Fully modular: agent, client, model, and tools.
- No cloud/OpenAI needed.

## Requirements

- torch
- transformers
- (optional) vllm
- pyyaml

Edit `configs/model_config.yaml` for model path and device.
